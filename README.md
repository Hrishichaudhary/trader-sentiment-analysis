# Trader Behavior vs Market Sentiment Analysis

## Objective
Analyze the relationship between trader performance and market sentiment to uncover behavioral patterns and derive strategy-relevant insights.

## Datasets
- Historical trader data
- Bitcoin Fear & Greed Index

## Approach
- Data preprocessing and sentiment alignment
- Feature engineering for performance and risk metrics
- Sentiment-wise analysis of trader behavior
- Visualization-driven insight discovery

## Key Insights
- Trader performance improves during Greed and Extreme Greed regimes.
- Greed periods exhibit higher tail risk despite better average performance.
- Profitable traders adapt better to sentiment shifts than losing traders.
- Risk behavior varies significantly across sentiment regimes.

## Tools
Python, Pandas, Matplotlib, Seaborn
